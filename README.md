# Site-Vitrine

## Description
Mon site vitrine exposant la plupart de mes projets.

## Fonctionnalités
- **Présentation de projets** : Découvrez les différents projets sur lesquels j'ai travaillé.
- **Contact** : Formulaire pour me contacter directement.

## Technologies utilisées
- **Front-end** : HTML, CSS, JavaScript
- **Back-end** : PHP
- **Base de données** : MySQL

## Installation

### Prérequis
- Un serveur web compatible avec PHP (par exemple, Apache)
- MySQL installé sur le serveur
- Un outil pour gérer les bases de données, tel que phpMyAdmin
